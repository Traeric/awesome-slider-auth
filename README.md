# awesome-slider-auth

> 1.x.x的版本都是测试版 正式版本从2.0.1开始

## 简介
这是一个使用VUE3.0编写的人机校验组件库 目的是为了解决登录验证时识别人机操作的问题 所支持的组件如下:
* 滑动拼图
* 滑动认证
## 版本
支持VUE3+
## 下载
```javascript
npm i awesome-slider-auth
```
